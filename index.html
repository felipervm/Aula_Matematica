MMC e MDC

Revisão interativa passo a passo

Vamos revisar juntos!

Hoje você vai entender de vez a diferença entre MDC e MMC — e no final a gente resolve uma questão de prova real das Olimpíadas.

Qual é a diferença?
MDC — Máximo Divisor Comum
É o maior número que divide os dois ao mesmo tempo, sem sobrar nada.
Pensa: o maior "pedaço" que cabe nos dois.
MMC — Mínimo Múltiplo Comum
É o menor número que é múltiplo dos dois ao mesmo tempo.
Pensa: a menor "tabela" onde os dois se encontram.
Como lembrar qual é qual?
D
MDC = Divisor → divide os números. O resultado é sempre menor ou igual aos números.
M
MMC = Múltiplo → é múltiplo dos números. O resultado é sempre maior ou igual aos números.
Quando usar cada um?

MDC

Quando quer dividir em partes iguais sem sobrar.

"Qual o maior pedaço?"

MMC

Quando quer saber quando dois eventos coincidem de novo.

"Quando vai repetir junto?"

MDC — Máximo Divisor Comum

O maior número que divide os dois ao mesmo tempo, sem sobrar nada.

Exemplo visual: MDC(12, 18)

Divisores de cada número:

12: 123 4612
18: 123 6918
Divisores comuns: 1, 2, 3, 6
O maior deles é o 6.
Logo: MDC(12, 18) = 6 ✅

Teste rápido
Qual é o MDC(8, 12)?

Dica: 8 = 2³ · 12 = 2² × 3 → fator comum: 2², menor expoente = 4

MMC — Mínimo Múltiplo Comum

O menor número que é múltiplo dos dois ao mesmo tempo.

Exemplo visual: MMC(4, 6)

Múltiplos de cada número:

4: 4812 162024
6: 612 182430
Múltiplos comuns: 12, 24, 36…
O menor é o 12.
Logo: MMC(4, 6) = 12 ✅
